## How to collaberate

1. Clone the repository:

```
 git clone https://github.com/yourusername/yourproject.git
```

2. Pull the repo:

```
  git pull
```

3. Checkout a new branch:

```
 git checkout -b branch-name
```

4. Add and commit:

```
git add .
git commit -am "message"
```

5. Push changes

```
git push
```

6. Press Compare & Pull request in github
